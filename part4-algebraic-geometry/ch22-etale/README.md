# 第二十二章：平展上同调初步

> **平展上同调**是代数几何中极其重要的上同调理论，特别是在数论几何中。

---

## 22.1 平展态射

### 22.1.1 定义

> **定义 22.1.1（平展态射）**
> 
> 态射 f: X → Y 是**平展的（Étale）**，如果：
> - 局部自由
> - 泛开（generic fiber is smooth）

```text
例子：
- 开浸入是平展态射
- 有限平展扩张
```

### 22.1.2 例子

```text
例子1：圆周 S¹ → A¹
- 不是平展

例子2：P¹ 的两次覆盖
- 有理函数 t → t²
- 是平展（局部同构）
```

---

## 22.2 平展拓扑

### 22.2.1 平展拓扑

> **定义 22.2.1（平展拓扑）**
> 
> 平展拓扑的开集是平展映射的像：
> 
> ```
> Y → X 局部同构
> ```

### 22.2.2 平展覆盖

```text
例子：
- 在 X 上选取平展覆盖
- 形成 Grothendieck 拓扑
```

---

## 22.3 平展上同调

### 22.3.1 定义

> **定义 22.3.1（平展上同调）**
> 
> **平展上同调** H^•_ét(X, F) 是：
> - 在平展 site 上计算的层上同调
> - 与拓扑上同调相似

### 22.3.2 例子

```text
例子：对有限域
- H^i_ét(X, ℤ/n) 可以计算
- 与 l-adic 上同调关联
```

---

## 22.4 l-adic 上同调

### 22.4.1 l-adic 层

> **定义 22.4.1（l-adic 层）**
> 
> **l-adic 层**是：
> 
> ```
> ℤ/l^k → ℤ/l^{k+1}
> ```
> 
> 的逆向系。

### 22.4.2 定义

```text
H^i_ét(X, ℚ_l) = lim H^i_ét(X, ℤ/l^k) ⊗ ℚ_l
```

---

## 22.5 应用

### 22.5.1 Weil 猜想

> **应用（Weil 猜想）**
> 
> 通过 l-adic 上同调可以证明 Weil 猜想：
> - ζ 函数的函数方程
- 黎曼假设类比

### 22.5.2 数论应用

```text
- 朗兰兹纲领
- 伽罗瓦表示
```

## 22.6 小结

| 概念 | 描述 |
