# 第二十八章：超曲面与射影几何进阶

> 本章深入探讨超曲面的几何性质。

---

## 28.1 超曲面的定义

### 28.1.1 定义

> **超曲面**是射影空间中由单个齐次方程定义的子簇：
> 
> ```
> X = V_+(f) ⊂ P^n
> ```
> 
> 其中 f 是 d 次齐次多项式。

### 28.1.2 次数与维数

```text
- 维数 = n - 1
- 次数 = d
```

---

## 28.2 消逝定理

### 28.2.1 消逝定理

> **消逝定理**：对丰富层 L，有：
> 
> ```
> H^i(P^n, L) = 0，对 i > 0
> ```

### 28.2.2 应用

```text
- 可逆层的截面计算
- 嵌入维度估计
```

---

## 28.3 典范嵌入

### 28.3.1 嵌入维度

> 对超曲面 X ⊂ P^n：
> 
> - 当 d ≥ 2n-1 时，X 是线性正规
> - 典范映射是嵌入

### 28.3.2 例子

```text
例子：
- 平面曲线（n=2）
- 曲面（n=3）
```

---

## 28.4 小结

| 概念 | 描述 |
|------|------|
| 超曲面 | 单个方程定义的子簇 |
| 消逝定理 | 丰富层上同调 vanish |
| 线性正规 | 嵌入性质 |
