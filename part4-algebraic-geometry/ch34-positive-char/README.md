# 第三十四章：特征p几何初步

> 本章介绍特征p代数几何的基础知识。

---

## 34.1 特征p基础

### 34.1.1 域的特征

> **特征** p > 0 的域满足：
> 
> ```
> 1 + 1 + ... + 1 (p次) = 0
> ```

### 34.1.2 例子

```text
例子：
- 𝔽_p = ℤ/pℤ
- 𝔽_p(t) = 有理函数域
- 代数闭包 𝔽̄_p
```

---

## 34.2 Frobenius

### 34.2.1 定义

> **Frobenius 态射**是：
> 
> ```
> F: X → X, x ↦ x^p
> ```

### 34.2.2 性质

```text
- F 是幂零的（在某些情况下）
- 与 p 可除性相关
```

---

## 34.3 曲面在特征p

### 34.3.1 特殊性质

- p 可除曲面的新现象
- 奇点解消

### 34.3.2 例子

```text
例子：
- 超曲面在特征 p
- Artin-Schreier 覆盖
```

## 34.4 小结

| 概念 | 描述 |
|------|------|
| 特征p | p×1=0 的域 |
| Frobenius | x→x^p 映射 |
| p可除 | 与p相关的几何 |
