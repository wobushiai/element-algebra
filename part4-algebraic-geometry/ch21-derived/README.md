# 第二十一章：导出范畴初步

> **导出范畴**是现代代数几何的核心工具。本章介绍导出范畴的基本概念。

---

## 21.1 导出范畴的动机

### 21.1.1 为什么要导出范畴？

> **问题**：层上同调不是正合函子
> 
> **解决**：把层「消解」成复形，然后用复形的同调来定义上同调！

### 21.1.2 基本思想

```text
层 F
  ↓消解
复形 I^• （每个 I^j 是内射层）
  ↓取截面
Γ(X, I^•)
  ↓取同调
H^j(X, F)
```

---

## 21.2 复形

### 21.2.1 复形的定义

> **定义 21.2.1（复形）**
> 
> **复形** C^• 是带有微分 d: C^j → C^{j+1} 的对象序列，满足 d² = 0。

```text
例子：
C^0 → C^1 → C^2 → ...
d        d        d
C^0 ← C^1 ← C^2 ← ...
```

### 21.2.2 链复形与上链复形

```text
- 链复形：C_n → C_{n-1}
- 上链复形：C^n → C^{n+1}
- 本书主要用上链复形
```

---

## 21.3 导出范畴的定义

### 21.3.1 同伦范畴

> **定义 21.3.1（同伦）**
> 
> 两个态射 f, g: C^• → D^• 是**同伦**的，如果存在同伦算子 h 使得：
> 
> ```
> f - g = d h + h d
> ```

### 21.3.2 导出范畴

> **定义 21.3.2（导出范畴）**
> 
> **导出范畴** D(X) 是：
> - 对象：复形（有界/有界 below/无界）
> - 态射：在同伦范畴中局部化

---

## 21.4 导出函子

### 21.4.1 右导出函子

> **定义 21.4.1（右导出函子）**
> 
> 对函子 F，右导出函子 R^j F 是：
> 
> ```
> R^j F(C) = H^j(F(I^•))
> ```
> 
> 其中 I^• → C 是内射消解。

### 21.4.2 例子

```text
例子：层上同调
- R^jΓ(X, -) = H^j(X, -)
- 这正是我们定义的层上同调！
```

---

## 21.5 三角范畴

### 21.5.1 三角的定义

> **定义 21.5.1（三角）**
> 
> 导出范畴是**三角范畴**，带有三角：
> 
> ```
> A → B → C → A[1]
> ```

### 21.5.2 性质

```text
- 导出范畴的同构类更丰富
- 可以做「傅里叶-Mukai 变换」
- 等等...
```
