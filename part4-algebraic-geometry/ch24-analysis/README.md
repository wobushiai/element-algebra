# 第二十四章：热方程与几何分析初步

> 本章介绍与代数几何相关的几何分析概念。

---

## 24.1 调和映照

### 24.1.1 定义

> **定义 24.1.1（调和映照）**
> 
> 映照 f: M → N 是**调和的**，如果它是能量泛函的临界点：
> 
> ```
> E(f) = ∫ |df|² dV
> ```

### 24.1.2 与代数几何的联系

```text
- 极小曲面
- 复曲线（局部是调和映照）
```

---

## 24.2 复 Monge-Ampère 方程

### 24.2.1 定义

> **定义 24.2.1（复 Monge-Ampère）**
> 
> **复 Monge-Ampère 方程**是：
> 
> ```
> (dd^c φ)ⁿ = e^φ dV
> ```

### 24.2.2 应用

```text
- Kähler-Einstein 度量
- 复几何中的唯一性
```

---

## 24.3 几何流

### 24.3.1 Ricci 流

> **定义 24.3.1（Ricci 流）**
> 
> **Ricci 流**是：
> 
> ```
> ∂g/∂t = -Ric(g)
> ```

### 24.3.2 与代数几何

```text
- Kähler-Ricc 流
- 奇点分析
```

---

## 24.4 稳定曲线

### 24.4.1 稳定性的定义

> **定义 24.4.1（K-稳定性）**
> 
> **K-稳定性**是：
> - 极小化 Chow 归一化能量
> - 与 Kähler-Einstein 度量等价

### 24.4.2 应用

```text
- 模空间的紧化
- 几何不变式理论
```
