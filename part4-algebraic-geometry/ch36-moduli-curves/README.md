# 第三十六章：曲线模空间进阶

## 36.1 Teichmüller 空间

### 36.1.1 定义

> **Teichmüller 空间** T_g 是：
> - 亏格 g 曲线的模空间
> - 同构于 3g-3 维开球

### 36.1.2 例子

```text
例子：
- T_1 同构于 punctured disc
- T_2 是 complex ball
```

---

## 36.2 曲线族

### 36.2.1 稳定性

> **稳定曲线**是：
> - 仅有节点奇点
> - 有限自同构

### 36.2.2 紧化

```text
- Deligne-Mumford 紧化
- 添加的边界对应退化曲线
```

## 36.3 交点理论

### 36.3.1 基本类

> **曲线模空间**带有类：
> - λ 类：Hodge 层类
> - δ 类：边界类

### 36.3.2 公式

```text
- Harer-Zagier 公式
- 欧拉示性数计算
```
