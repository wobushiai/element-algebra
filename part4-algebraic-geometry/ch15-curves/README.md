# 第十五章：曲线论基础 —— 一维几何的世界

> **本章导读**：代数曲线是代数几何中最丰富、最深入研究的对象之一。从有理曲线到椭圆曲线，曲线理论展示了代数几何的核心思想和方法。这一章会详细介绍曲线的基本理论。

---

## 15.1 曲线的定义

### 15.1.1 什么是代数曲线？

> **定义 15.1.1（代数曲线）**
> 
> **代数曲线（Algebraic Curve）** 是：
> - 1 维的
> - 有限型的
> - 分离的
> 
> 概形（通常定义在代数闭域上）。

```text
等价定义：
- 曲线 = 维数为 1 的簇（variety）
- 曲线 = 射影空间中由齐次多项式定义的超曲面（余维1）
- 曲线 = 函数域 K/k 是 1 次超越扩张
```

### 15.1.2 曲线的例子

```text
例子1：有理曲线 P¹
- 最简单的曲线
- 函数域 ≅ k(t)
- 亏格 g = 0

例子2：椭圆曲线
- 亏格 g = 1
- 函数域是 2 次扩张
- 有群结构！

例子3：一般平面曲线
- d 次平面曲线：C: F(x,y,z) = 0
- 亏格 g = (d-1)(d-2)/2
- d=3: g=1（椭圆曲线）
- d=4: g=3
```

## 15.2 曲线的分类

### 15.2.1 按亏格分类

> **定义 15.2.1（亏格）**
> 
> 曲线 C 的**亏格（Genus）** g 是：
> - 拓扑亏格（黎曼面观点）
> - 几何亏格（代数几何定义）
> - 算术亏格（pₐ = (d-1)(d-2)/2）

```text
分类：
- g = 0：有理曲线（同构于 P¹）
- g = 1：椭圆曲线（有群结构）
- g ≥ 2：一般曲线（模空间高维）
```

### 15.2.2 有理曲线

> **命题 15.2.2（有理曲线的刻画）**
> 
> 曲线 C 是有理的（g = 0）当且仅当：
> - 存在非常数态射 P¹ → C
> - 或者 C 同构于 P¹

```text
例子：
- P¹ 本身
- 直线
- 圆锥曲线（圆锥曲线都是有理的）
- 扭结三次曲线（有奇点的有理曲线）
```

### 15.2.3 椭圆曲线

> **定义 15.3.1（椭圆曲线）**
> 
> **椭圆曲线**是：
> - 亏格 g = 1 的
> - 光滑的
> - 带有指定基点（单位元）
> 
> 的曲线。

```text
典型方程（Weierstrass 形式）：
y²z = x³ + axz² + bz³
其中 4a³ + 27b² ≠ 0（确保光滑）

在仿射坐标中：
y² = x³ + ax + b
```

## 15.3 除子理论

### 15.3.1 Weil 除子

> **定义 15.3.1（Weil 除子）**
> 
> 曲线 C 上的 **Weil 除子**是形式和：
> 
> ```
> D = Σ n_i P_i
> ```
> 
> 其中 P_i 是闭点，n_i ∈ ℤ。

```text
例子：
- D = 3P（点在 P 处的 3 倍）
- D = P + Q + R（三个点的和）
- D = 0（零除子）
```

### 15 除子的度

> **定义 15.3.2（除子的度）**
> 
> 除子 D = Σ n_i P.3.2_i 的**度**定义为：
> 
> ```
> deg(D) = Σ n_i
> ```

```text
例子：
- deg(3P) = 3
- deg(P + Q + R) = 3
- deg(0) = 0
```

### 15.3.3 主除子

> **定义 15.3.3（主除子）**
> 
> 对有理函数 f ∈ k(C)*，**主除子**定义为：
> 
> ```
> div(f) = Σ ord_P(f) · P
> ```
> 
> 其中 ord_P(f) 是 f 在点 P 处的阶（零点的阶或极点的负阶）。

```text
例子：
- 在 P¹ 上：div(x) = 0·∞ + 1·0（实际上更复杂）
- 在椭圆曲线上：div(f) 的度总是 0
```

## 15.4 线丛与可逆层

### 15.4.1 可逆层

> **定义 15.4.1（可逆层/线丛）**
> 
> **可逆层（Invertible Sheaf）** 或 **线丛（Line Bundle）** 是局部同构于结构层的层。

```text
等价刻画：
- 可逆层 = 秩 1 的局部自由层
- 可逆层 = 秩 1 的 coherent 层
```

### 15.4.2 线丛与除子的对应

> **定理 15.4.2（除子 ↔ 线丛）**
> 
> 曲线 C 上的可逆层与 Weil 除子之间存在**一一对应**（模去主除子）：
> 
> ```
> Pic(C) / {主除子} ≅ {可逆层} / ≅
> ```

```text
几何意义：
- 每个除子 D 定义一个线丛 O(D)
- O(D) 的截面是在 D 外全纯、在 D 处有指定极点的函数
```

### 15.4.3 截面层

> **定义 15.4.3（截面层）**
> 
> 对除子 D，**截面层**定义为：
> 
> ```
> O_C(D)(U) = {f ∈ k(C)* | div(f) + D ≥ 0 on U} ∪ {0}
> ```

```text
例子：
- O(P¹) 的截面 = 常数函数
- O(dP) 的截面 = 在 P 处至多 d 阶极点的函数
- l(D) = dim H⁰(C, O(D))
```

## 15.5 Riemann-Roch 定理

### 15.5.1 定理的陈述

> **定理 15.5.1（Riemann-Roch）**
> 
> 对曲线 C 和除子 D：
> 
> ```
> l(D) - l(K - D) = deg(D) - g + 1
> ```
> 
> 其中：
> - g 是 C 的亏格
> - K 是典范除子（任何典范层的全纯除子）
> - l(D) = dim H⁰(C, O(D))

```text
解释：
- 左边：截面数 - 对偶截面数
- 右边：可计算的不变量
- 这给出了截面数的「公式」！
```

### 15.5.2 应用例子

```text
例子1：度数为 d 的除子
- 当 d ≥ 2g - 1 时：l(D) = d - g + 1

例子2：椭圆曲线（g = 1）
- l(D) = deg(D)（当 deg(D) ≥ 0）
- l(K - D) = 0（当 deg(D) ≥ 0）

例子3：计算典范层
- deg(K) = 2g - 2
- 对 g = 1：deg(K) = 0
- 对 g = 2：deg(K) = 2
```

## 15.6 椭圆曲线

### 15.6.1 椭圆曲线的群结构

> **定理 15.6.1（椭圆曲线的群结构）**
> 
> 椭圆曲线 E 上有一个**阿贝尔群结构**：
> - 单位元：基点 O
> - 加法：P + Q + R = 0（如果 P, Q, R 共线）
> - 逆元：-P 是 P 关于 x-轴的对称点

```text
几何解释：
- 三点共线 → 它们的和为零
- 这给出了「曲线上的加法」！
```

### 15.6.2 椭圆曲线的例子

```text
例子：y² = x³ + x
- 定义域：k（特征 ≠ 2）
- 群结构：
  - O = 点在无穷远
  - (x, y) + (x, -y) = O
  - (x₁, y₁) + (x₂, y₂) = (x₃, y₃)
```

## 15.7 曲线间的态射

### 15.7.1 有理映射

> **定义 15.7.1（有理映射）**
> 
> 曲线之间的**有理映射** φ: C → D 是：
> - 除子映射（除子之间的映射）
- 诱导函数域扩张 k(D) ⊆ k(C)

```text
重要事实：
- 曲线之间的任何有理映射都是态射
- （不存在「真正的」有理映射不是态射的情况！）
```

### 15.7.2 覆盖理论

> **定义 15.7.2（有限态射）**
> 
> 态射 f: C → C' 是**有限的**，如果：
> - 预像 of any point 是有限集
- 对应函数域是有限扩张

```text
例子：
- 次数 d 的覆盖 = 函数域的 d 次扩张
- 例如：P¹ 的 d 次分枝覆盖
```

## 15.8 曲线模空间初步

### 15.8.1 M_g

> **定义 15.8.1（曲线模空间）**
> 
> **M_g** 是亏格 g 的光滑曲线的模空间：
> - 对象：同构类 [C]
> - 态射：曲线之间的态射

```text
性质：
- dim M_g = 3g - 3（当 g ≥ 2）
- M_0 ≅ P¹
- M_1 更复杂（椭圆曲线模）
```

### 15.8.2 紧化

> **定理 15.8.2（Deligne-Mumford 紧化）**
> 
> 存在紧化：
> 
> ```
> M̄_g ⊇ M_g
> ```
> 
> 加入**稳定曲线**（允许节点奇点）。
