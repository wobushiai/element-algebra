# 第三十一章：代数几何中的组合学初步

> 本章介绍代数几何与组合学的交叉。

---

## 31.1 扇与锥

### 31.1.1 扇的定义

> **扇（Fan）**是锥的集合，满足：
> - 闭于取子锥
> - 每个锥可表示为有限个生成锥的并

### 31.1.2 例子

```text
例子：
- 仿射空间的扇
- toric varieties 的扇
```

---

## 31.2 Toric 簇

### 31.2.1 定义

> **Toric 簇**是从扇构造的代数簇：
> 
> ```
> X_Σ = ∪_σ O(σ)
> ```

### 31.2.2 例子

```text
例子：
- P^n
-  Hirzebruch 曲面
```

---

## 31.3 多面体几何

### 31.3.1 牛顿多面体

> **牛顿多面体**Newton(N_f) 是：
> 
> ```
> Conv{ (a₁,...,aₙ) | x₁^{a₁}...xₙ^{aₙ} 在 f 中出现 }
> ```

### 31.3.2 应用

```text
- 奇点解消
-  toric 几何
```

---

## 31.4 小结

| 概念 | 描述 |
|------|------|
| 扇 | 锥的组合收集 |
| Toric 簇 | 从扇构造的簇 |
| 牛顿多面体 | 多项式的组合对象 |
