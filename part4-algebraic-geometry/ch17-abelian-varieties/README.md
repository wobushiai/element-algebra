# 第十七章：Abel 簇初步 —— 高维阿贝尔群几何

> **Abel 簇是带有群结构的完整代数簇。它们是代数几何中最重要的对象之一，在数论、模空间和数学物理中有广泛应用。本章介绍Abel簇的基本理论。

---

## 17.1 Abel簇的定义

### 17.1.1 定义

> **定义 17.1.1（Abel簇）**
> 
> **Abel簇** A 是：
> 1. **完整的**（proper）：类似紧致
> 2. **光滑的**：没有奇点
> 3. **群结构**：带有阿贝尔群运算
> 
> 的代数簇。

```text
例子：
- 椭圆曲线（g=1的Abel簇）
- 雅可比簇（曲线的参数化）
- 一般g维Abel簇
```

### 17.1.2 基本性质

> **性质 17.1.2（Abel簇的基本性质）**
> 
> - 阿贝尔簇是交换群
> - 任何阿贝尔簇是射影的（可嵌入到射影空间）
> - Abel簇的子群仍是Abel簇

## 17.2 Abel簇的例子

### 17.2.1 椭圆曲线

> **例子 17.2.1（椭圆曲线）**
> 
> 椭圆曲线是最简单的非平凡Abel簇（g=1）。

```text
E: y² = x³ + ax + b
- 群结构：几何加法
- 维数：1
```

### 17.2.2 雅可比簇

> **例子 17.2.2（雅可比簇）**
> 
> 对任何曲线C（g≥1），存在一个g维Abel簇Jac(C)：

```text
定义：
- Jac(C) = Pic^0(C) = 除子类群
- 维数 = g
- 参数化了度为0的除子等价类

例子：
- Jac(P¹) = 0（退化）
- Jac(椭圆曲线) ≅ 椭圆曲线本身
- 对g≥2的曲线，Jac(C)是新的Abel簇
```

### 17.2.3 乘积Abel簇

> **例子 17.2.3（乘积）**
> 
> Abel簇的乘积仍是Abel簇：
> 
> ```
> A × B 是 g_A + g_B 维Abel簇
> ```

## 17.3 极化

### 17.3.1 极化层的定义

> **定义 17.3.1（极化）**
> 
> Abel簇A上的**极化**是一个等价类[λ]，其中λ: A → A^t是对偶Abel簇的同态，由一个 ample 可逆层L定义。

```text
等价定义：
- 极化 = 来自丰富层的同态
- 维数1：完全极化
- 高维：部分极化
```

### 17.3.2 主极化

> **定义 17.3.2（主极化）**
> 
> 主极化是度为1的极化：
> 
> ```
> deg λ = 1
> ```

```text
例子：
- 椭圆曲线上的主极化来自O(P)
- 雅可比簇Jac(C)带有主极化（来自θ-除子）
```

## 17.4 态射与同源

### 17.4.1 同态

> **定义 17.4.1（Abel簇同态）**
> 
> Abel簇之间的态射如果是群同态，则称为**同态**。

```text
性质：
- Hom(A, B) 是有限生成自由阿贝尔群
- 同态的核是Abel簇的子簇
```

### 17.4.2 同源

> **定义 17.4.2（同源）**
> 
> 两个Abel簇 A 和 B 是**同源的**，如果存在有限态射 A → B：

```text
同源关系：
- 保持维数
- 椭圆曲线都同源于P¹（false）
- 实际上：椭圆曲线形成自己的同源类
```

## 17.5 复Abel簇

### 17.5.1 作为复环面

> **定理 17.5.1（复Abel簇）**
> 
> 任何复Abel簇同构于：
> 
> ```
> A ≅ V/Λ
> ```
> 
> 其中V是C^g，Λ是格（rank 2g的Z-子空间）。

```text
例子：
- g=1: A = C/Λ = 椭圆曲线
- g=2: A = C²/Λ（2维复环面）
```

### 17.5.2 黎曼矩阵

> **定义 17.5.2（黎曼矩阵）**
> 
> 格Λ由基定义：
> 
> ```
> Λ = Zω₁ + Zω₂
> ```
> 
> 黎曼矩阵 τ = ω₂⁻¹ω₁ ∈ H_g（Siegel上半平面）

## 17.6 Abel簇的模空间

### 17.6.1 A_g

> **定义 17.6.1（Abel簇模空间）**
> 
> A_g是g维主极化Abel簇的模空间：

```text
性质：
- dim A_g = g(g+1)/2
- 这是 Siegel 紧化空间
```

### 17.6.2 紧化

> **定义 17.6.2（紧化）**
> 
> A_g可以紧化为：
> 
> ```
> \bar{A}_g = A_g ∪ 边界
> ```
> 
> 边界对应于退化的Abel簇。
