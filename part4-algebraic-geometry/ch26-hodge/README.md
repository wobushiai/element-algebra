# 第二十六章：Hodge 理论初步

> **Hodge 理论**是连接代数几何、拓扑学和复分析的强大工具。

---

## 26.1 Hodge 分解

### 26.1.1 基本定义

> **定义 26.1.1（Hodge 分解）**
> 
> 对复流形 X，其上同调有分解：
> 
> ```
> H^k(X, ℂ) = ⊕_{p+q=k} H^{p,q}
> ```
> 
> 其中 H^{p,q} 是 (p,q) 型微分形式。

### 26.1.2 例子

```text
例子：对复曲面 (dim = 2)
H² = H^{2,0} ⊕ H^{1,1} ⊕ H^{0,2}
```

## 26.2 Kähler 流形

### 26.2.1 定义

> **定义 26.2.1（Kähler 流形）**
> 
> **Kähler 流形**是带有 Kähler 度量的复流形。

### 26.2.2 性质

```text
- 复流形 + Hermite 度量 + 闭形式
- 射影簇都是 Kähler 的
```

## 26.3 Hodge 结构

### 26.3.1 定义

> **定义 26.3.1（Hodge 结构）**
> 
> **Hodge 结构**是带有分次的过滤结构。

### 26.3.2 应用

```text
- 周期映射
- 模空间理论
```
