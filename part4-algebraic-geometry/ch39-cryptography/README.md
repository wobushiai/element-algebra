# 第三十九章：代数几何在密码学中的应用

> 本章介绍代数几何在现代密码学中的应用。

---

## 39.1 椭圆曲线密码学

### 39.1.1 基础

> **椭圆曲线密码学（ECC）**基于：
> - 椭圆曲线上的离散对数问题
> - 相比 RSA 更短的密钥

### 39.1.2 标准曲线

```text
例子：
- P-256
- Curve25519
- secp256k1
```

---

## 39.2 超椭圆曲线密码学

### 39.2.1 超椭圆曲线

> **超椭圆曲线**是亏格 g ≥ 2 的曲线：
> 
> ```
> y² = f(x)
> ```

### 39.2.2 例子

```text
例子：
- g=2: y² = x^5 + ax + b
- 密码学应用
```

---

## 39.3 同源密码学

### 39.3.1 同源

> **同源密码学**使用：
> - 曲线间的同源
> - 隐藏曲线结构

### 39.3.2 应用

```text
- 隐私保护
- 零知识证明
```

---

## 39.4 后量子密码

### 39.4.1 抗量子

> **格密码**是后量子候选：
> - 基于格理论
> - 与代数几何相关

### 39.4.2 例子

```text
- NTRU
- Ring-LWE
```

---

## 39.5 小结

| 概念 | 描述 |
|------|------|
| ECC | 椭圆曲线密码 |
| 超椭圆曲线 | g≥2曲线密码 |
| 同源 | 基于同源的密码 |
| 后量子 | 抗量子攻击 |
