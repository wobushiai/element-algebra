## 42.2 傅里叶-穆卡伊变换

### 42.2.1 定义

> **傅里叶-穆卡伊（FM）变换**是：
> 
> ```
> Φ_P(E) = p_*(E ⊗ P)
> ```

### 42.2.2 应用

```text
- 派生等价的构造
- 曲线的模空间
```
