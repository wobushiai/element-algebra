## 41.1 相交类

### 41.1.1 定义

> **相交类**在周环 A^*(X) 中：
> - 闭合链的有理等价类
> - 乘法结构

### 41.1.2 例子

```text
例子：
- 点：类 = 1
- 曲线：类 = c
- 曲面：类 = s
```

---

## 41.2 黎曼-罗赫

### 41.2.1 定理

> **黎曼-罗赫**：
> 
> ```
> χ(X, F) = deg(ch(F) · td(T_X))
> ```

### 41.2.2 应用

```text
- 曲线的 Riemann-Roch
- 曲面的 Noether 公式
```

---

## 41.3 例子

### 41.3.1 平面曲线

```text
- 直线与直线：1点
- 直线与曲线：d点（d=次数）
- 曲线与曲线：d·e点
```
