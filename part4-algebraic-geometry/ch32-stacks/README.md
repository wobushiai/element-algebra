# 第三十二章：代数叠与模问题初步

> 本章介绍更高级的模空间理论。

---

## 32.1 叠的精确定义

### 32.1.1 定义

> **代数叠**是满足以下条件的范畴：
> - 是 sheaf of groupoids
> - 局部同构于某个代数对象的分类空间

### 32.1.2 例子

```text
例子：
- M_g：曲线模叠
- Bun_G：主丛模叠
```

---

## 32.2 稳定曲线

### 32.2.1 定义

> **稳定曲线**是：
> - 只有节点奇点
> - 有限自同构

### 32.2.2 紧化

```text
M_g 的 Deligne-Mumford 紧化：
M_g ⊂ \bar{M}_g
```

## 32.3 几何不变式理论

### 32.3.1 商

> **GIT 商**给出：
> 
> ```
> X // G = Proj(⊕_{n≥0} Γ(X, L^⊗n)^G)

### 32.3.2 应用

- 商簇
