# 第十八章：模空间初步

> **模空间**是代数几何的核心概念之一，它参数化了某一类代数对象的同构类。本章介绍模空间的基本思想。

---

## 18.1 什么是模空间？

### 18.1.1 动机

> **问题**：如何参数化所有亏格为 g 的光滑曲线？

这就是**曲线模空间** M_g 要做的事情！

### 18.1.2 定义

> **定义 18.1.1（粗模空间）**
> 
> **粗模空间** M 参数化某类代数对象的同构类：
> 
> - 对象：{X}
> - 同构类：X/～
> - 函子：T → {T-族}/～

```text
例子：
- M_g：g亏格曲线的模空间
- A_g：g维主极化Abel簇的模空间
- M_{0,n}：n标记点的球面模空间
```

---

## 18.2 曲线模空间

### 18.2.1 M_g

> **定义 18.2.1（曲线模空间）**
> 
> M_g 是亏格 g 的光滑曲线同构类的参数空间：
> 
> - 维数：3g - 3（g ≥ 2）
> - g = 0: M_0 ≅ P¹
> - g = 1: M_1 更复杂

### 18.2.2 Deligne-Mumford紧化

> **定义 18.2.2（稳定曲线）**
> 
> **稳定曲线**是：
> - 仅有节点奇点
> - 有限自同构

> **紧化**：
> 
> ```
> M̄_g ⊇ M_g
> ```
> 
> 加入稳定曲线。

---

## 18.3 希尔伯特模空间

### 18.3.1 希尔伯特函子

> **定义 18.3.1（希尔伯特多项式）**
> 
> 对射影簇 X ⊂ Pⁿ，其**希尔伯特多项式**是：
> 
> ```
> P(m) = χ(O_X(m))
> ```

### 18.3.2 希尔伯特概形

> **定义 18.3.2（希尔伯特概形）**
> 
> 希尔伯特概形参数化了：
> - 固定 Hilbert 多项式 P
> - 嵌入到射影空间的子簇

## 18.4 叠

### 18.4.1 堆的定义

> **定义 18.4.1（代数叠）**
> 
> **代数叠**是泛函问题可表的层范畴。

```text
例子：
- 曲线叠：参数化曲线的「层」
- 阿贝尔簇叠
- 带结构的曲线叠
```

### 18.4.2 例子

```text
例子：M_{1,1}（带一个标记点的椭圆曲线）
- 这是一个1维叠
- 同构于一条椭圆曲线（作为模空间）
```

---

## 18.5 局部结构

### 18.5.1 形式邻域

> **定义 18.5.1（形式邻域）**
> 
> 模空间在某点 X 的**形式邻域**：
> 
> ```
> \hat{M}_{[X]} ≅ Spf(Ĉ_{M,[X]})
> ```
> 
> 这给出了局部参数化。

### 18.5.2 切空间

> **定义 18.5.2（切空间）**
> 
> 模空间在点 [X] 的**切空间**是：
> 
> ```
> T_{[X]} M ≅ H¹(X, TX)
> ```
> 
> 其中 TX 是 X 的切层。
