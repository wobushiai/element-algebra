# 第三十三章：代数几何中的同调代数初步

> 本章介绍同调代数在代数几何中的应用。

---

## 33.1 复形与同调

### 33.1.1 基础回顾

> **复形**是带微分 d: C^j → C^{j+1} 的序列，满足 d² = 0。

### 33.1.2 同调群

```text
H^j(C^•) = Ker(d^{j+1}) / Im(d^j)
```

---

## 33.2 消解

### 33.2.1 各种消解

> **消解**用于计算导出函子：
> - 自由消解
> - 平坦消解
> - 内射消解

### 33.2.2 例子

```text
例子：
- Koszul 复形
- Čech 复形
- Godement 消解
```

---

## 33.3 Ext 与 Tor

### 33.3.1 Ext

> **Ext**是 Hom 的右导出：
> 
> ```
> Ext^i_R(M, N) = H^i(Hom_R(P^•, N))
> ```

### 33.3.2 Tor

> **Tor**是张量积的左导出：
> 
> ```
> Tor_i^R(M, N) = H_i(M ⊗_R P^•)
> ```

---

## 33.4 应用

### 33.4.1 在代数几何中

- 基变换

```text
例子：

```
