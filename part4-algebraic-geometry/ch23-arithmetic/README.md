# 第二十三章：算术几何初步

> **算术几何**是代数几何与数论的交叉领域。本章介绍算术几何的基本概念。

---

## 23.1 数域上的代数几何

### 23.1.1 概形 over ℤ

> **定义 23.1.1（算术曲面）**
> 
> **算术曲面**是定义在 ℤ 上的二维概形。

```text
例子：
- P¹_ℤ = Proj ℤ[x₀, ..., xₙ]
- 椭圆曲线 over ℤ
```

### 23.1.2 特殊化

```text
对每个素数 p：
- 模 p 约化得到有限域上的曲线
- 可能改变奇点类型
```

---

## 23.2 局部域

### 23.2.1 p 进数

> **定义 23.2.1（p 进数）**
> 
> **p 进数** ℚ_p 是 ℚ 的 p 进完备化。

### 23.2.2 局部域

```text
- ℚ_p, ℝ 是局部域
- 有限域的函数域也是局部域
```

---

## 23.3 整数环

### 23.3.1 代数整数环

> **定义 23.3.1（代数整数环）**
> 
> 数域 K 的**整数环** O_K 是 K 中代数整元的环。

```text
例子：
- ℤ = O_ℚ
- ℤ[i] = O_{ℚ(i)}
- ℤ[(1+√5)/2] = O_{ℚ(√5)}
```

### 23.3.2 理想分解

```text
在 O_K 中：
- 素理想分解
- 分歧指数
- 剩余类域度
```

---

## 23.4 曲线算术

### 23.4.1 有理点

> **定义 23.4.1（有理点）**
> 
> 曲线 C over 域 K 的**有理点**是：
> 
> ```
> C(K) = Hom(Spec K, C)
> ```

### 23.4.2 Mordell 猜想

> **定理 23.4.2（Mordell-Weil 定理）**
> 
> 椭圆曲线的有理点群是有限生成的：
> 
> ```
> E(ℚ) ≅ E(ℚ)_tors × ℤ^r
> ```

---

## 23.5 局部-整体原则

### 23.5.1 Hasse 原理

> **定义 23.5.1（Hasse 原理）**
> 
> 如果方程在每个局部域有解，则在全局域有解。

### 23.5.2 例子

```text
例子：二次型
- Hasse 原理成立
- 需要检查所有局部域
```
