## 37.1 阿贝尔簇的同源

### 37.1.1 同源群

> **同源群**是：
> 
> ```
> Hom(A, B) ⊗ ℚ
> ```
> 
> 同源类由同源群分类。

### 37.1.2 例子

```text
例子：
- 椭圆曲线同源 = 1维
- 一般g维阿贝尔簇同源 = g(g+1)/2 维
```
