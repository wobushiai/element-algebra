# 第二十五章：模形式与几何初步

> **模形式**是数论中的重要对象，与代数几何密切相关。

---

## 25.1 模形式

### 25.1.1 定义

> **定义 25.1.1（模形式）**
> 
> **模形式**是满足函数方程的复函数：
> 
> ```
> f(az+b/cz+d) = (cz+d)^{2k} f(z)
> ```
> 
> 其中 γ ∈ SL(2, ℤ)。

### 25.1.2 权

```text
- k 是权（weight）
- 不同权对应不同表示
```

---

## 25.2 椭圆曲线与模形式

### 25.2.1 模性

> **定理 25.2.1（温安霖-泰勒）**
> 
> 每个椭圆曲线 over ℚ 是**模的**：
> 
> ```
> E ↔ 权2 模形式
> ```

### 25.2.2 费马大定理

```text
- 椭圆曲线的模性
- 关键应用于 FLT 证明
```

## 25.3 Siegel 模形式

### 25.3.1 定义

> **定义 25.3.1（Siegel 上半平面）**
> 
> **Siegel 上半平面**是：
> 
> ```
> H_g = {g×g 复矩阵 | 对称正定}
> ```

### 25.3.2 应用

```text
- Siegel 模形式
- 阿贝尔簇模形式
```
