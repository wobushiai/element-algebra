# 第二十章：曲面论初步

> **曲面**是二维的代数簇，在代数几何中占有重要地位。本章介绍代数曲面的基本理论。

---

## 20.1 代数曲面的定义

### 20.1.1 定义

> **定义 20.1.1（代数曲面）**
> 
> **代数曲面**是维数为 2 的代数簇。

```text
例子：
- 射影平面 P²
- 二次曲面 P¹ × P¹
-  Hirzebruch 曲面
- K3 曲面
```

### 20.1.2 分类

> **分类 20.1.2（曲面分类）**
> 
> - 有理曲面：同构于 P² 的某个吹开
- K3 曲面：典范层平凡
- 椭圆曲面：带有椭圆纤维
- 一般型曲面：一般理论

---

## 20.2 有理曲面

### 20.2.1 有理曲面的定义

> **定义 20.2.1（有理曲面）**
> 
> **有理曲面**是同构于 P² 的某个吹开的曲面。

```text
例子：
- P² 本身
-  Hirzebruch 曲面 Σ_n
```

### 20.2.2 吹开

> **定义 20.2.2（吹开）**
> 
> **吹开（ Blow-up）** 是曲面的基本操作：
> - 去掉一个点
> - 以 P¹ 填充

---

## 20.3 K3 曲面

### 20.3.1 定义

> **定义 20.3.1（K3 曲面）**
> 
> **K3 曲面**是：
> - 光滑射影曲面
> - 典范层 O(K) ≅ O
> - 通用覆叠是 K3 曲面

### 20.3.2 例子

```text
例子：四次曲面
- 在 P³ 中由四次方程定义
- x⁴ + y⁴ + z⁴ + w⁴ = 0
```

---

## 20.4 椭圆曲面

### 20.4.1 定义

> **定义 20.4.1（椭圆曲面）**
> 
> **椭圆曲面**是带有椭圆纤维的曲面：
> 
> ```
> f: X → C
> ```
> 
> 其中一般纤维是椭圆曲线。

### 20.4.2 Kodaira 分类

```text
椭圆曲面的纤维类型：
- I_n: 平稳纤维
- II, III, IV: 奇异纤维
- 其他类型...
```

---

## 20.5 相交理论

### 20.5.1 相交数

> **定义 20.5.1（相交数）**
> 
> 曲面上曲线的**相交数** C·D 是：
> 
> ```
> C·D = χ(O_C) + χ(O_D) - χ(O_{C∩D})
> ```

### 20.5.2 例子

```text
例子：在 P² 中
- 直线 · 直线 = 1
- 直线 · 二次曲线 = 2
- 二次曲线 · 二次曲线 = 4
```
