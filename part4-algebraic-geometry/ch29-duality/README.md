# 第二十九章：Grothendieck 对偶性

> **Grothendieck 对偶性**是层上同调的基本定理。

---

## 29.1 导出范畴中的对偶

### 29.1.1 基本思想

> **对偶性**给出：
> 
> ```
> RHom(F, G) ≅ RHom(G^∨, F^∨)
> ```

### 29.1.2 例子

```text
例子：
- 对 X = P^n，对偶很简单
- 对一般簇，需要更多工具
```

---

## 29.2 局部对偶

### 29.2.1 局部版本

> **局部对偶**：对局部环 (R, m)：
> 
> ```
> Ext^i_R(k, M) ≅ Ext^{dim R-i}_R(M, R)^∨
> ```

### 29.2.2 应用

```text
- 局部上同调计算
- 奇点研究
```

---

## 29.3 Serre 对偶

### 29.3.1 射影簇

> **Serre 对偶**对射影簇 X：
> 
> ```
> H^i(X, F) ≅ H^{n-i}(X, F ⊗ ω_X)^∨
> ```

### 29.3.2 例子

```text
例子：
- P^n 的对偶
- 射影曲线的对偶
```
