# 第三十八章：代数几何中的射影几何进阶

> 本章深入探讨射影几何的高级理论。

---

## 38.1 射影嵌入

### 38.1.1 线性系统

> **线性系统**是：
> 
> ```
> |D| = P(H^0(X, O(D)))
> ```
> 
> 给出射影嵌入。

### 38.1.2 例子

```text
例子：
- 曲线的高阶嵌入
- 曲面的射影正规性
```

---

## 38.2 射影判别式

### 38.2.1 定义

> **射影判别式**是：
> - 判别超曲面
> - 与奇点相关

### 38.2.2 应用

```text
- 奇异轨迹
- 模空间边界
```

---

## 38.3 极化簇

### 38.3.1 极化

> **极化**是丰富层：
> 
> ```
> L: 丰富可逆层
> ```

### 38.3.2 例子

```text
- 射影空间 O(1)
- 曲线 O(dP)
```
