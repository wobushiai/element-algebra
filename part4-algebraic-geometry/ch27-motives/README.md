# 第二十七章：Motives 初步

> **Motives**（动机）被认为是「代数几何的上同调的终极统一理论」。

---

## 27.1 动机理论

### 27.1.1 基本思想

> **Motives** 是连接不同上同调理论的统一框架：
> 
> ```
>  variété → motive → cohomology
> ```

### 27.1.2 类别

```text
- Chow motives
- Pure motives
- Mixed motives
```

---

## 27.2  Chow 群

### 27.2.1 定义

> **Chow 群** CH^*(X) 是代数闭链的有理等价类群。

### 27.2.2 例子

```text
例子：
- CH^0(X) = ℤ（点的个数）
- CH^1(X) = Pic(X)
```

---

## 27.3 跨上同调

### 27.3.1 统一

> **跨上同调**是统一的 cohomology 理论：
> 
> - Betti cohomology
> - ℓ-adic cohomology  
> - de Rham cohomology
> - crystalline cohomology

---

## 27.4 小结

| 概念 | 描述 |
|------|------|
| Motives | 统一的 cohomology 理论 |
| Chow 群 | 代数闭链的等价类 |
| 跨上同调 | 统一不同上同调 |
