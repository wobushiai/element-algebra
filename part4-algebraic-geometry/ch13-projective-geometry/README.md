# 第十三章：射影空间与射影簇 —— 完美的无穷远点

> **本章导读**：射影几何是经典代数几何的核心。从「平行线在无穷远相交」这个直观想法出发，射影空间提供了一个「完备化」的几何世界。在这一章中，我们会非常详细地介绍射影空间的定义、它的几何性质，以及射影簇的基本理论。

---

## 13.1 动机：为什么要射影空间？

### 13.1.1 仿射空间的局限（详细解释）

**问题1：直线相交**
```text
在仿射平面 A² 中：
- 两条不同直线「通常」相交于一点
- 但**平行线不相交**！

例子：
- 直线 y = x + 1
- 直线 y = x + 2
- 这两条直线「平行」，永远不相交
```

**问题2：齐次坐标的优势**
```text
在仿射空间 A² 中：
- 点用坐标 (x, y) 表示
- 但「缩放」没有意义：(x, y) 和 (2x, 2y) 是不同的点

在射影空间 P² 中：
- 点用齐次坐标 [x : y : z] 表示
- [x : y : z] = [λx : λy : λz] 对任何非零 λ
- 这样「缩放」不改变点！
```

### 13.1.2 射影空间的直观理解

> **核心思想**：射影空间 = 「添加了无穷远点」的仿射空间

```text
直观理解：
- A² 中的每条直线都有一个「无穷远点」
- 所有无穷远点连成一条「无穷远直线」
- P² = A² ∪ ∞
```

## 13.2 射影空间的定义

### 13.2.1 作为商空间

> **定义 13.2.1（射影空间的商定义）**
> 
> 设 k 是代数闭域。**n 维射影空间** Pⁿ(k) 或简记为 Pⁿ 定义为：
> 
> ```
> Pⁿ = (k^{n+1} \ {0}) / ~
> ```
> 
> 其中等价关系 ∼ 定义为：
> 
> ```
> (x₀, x₁, ..., xₙ) ∼ (λx₀, λx₁, ..., λxₙ)  对任何非零 λ ∈ k
> ```

```text
几何直觉：
- 从 k^{n+1} 中去掉原点
- 把所有非零向量「粘」在一起
- 每条通过原点的直线对应射影空间中的一个点

记号：
- 点 [x₀ : x₁ : ... : xₙ] 叫「齐次坐标」
- : 表示「比例」，[1:2] = [2:4] = [3:6]
```

### 13.2.2 作为 Proj

> **定义 13.2.2（射影空间的 Proj 定义）**
> 
> 射影空间也可以定义为：
> 
> ```
> Pⁿ = Proj(k[x₀, x₁, ..., xₙ])
> ```
> 
> 其中 k[x₀, x₁, ..., xₙ] 是 (n+1) 个变量的多项式环，**Proj** 是某种「射影谱」的构造。

```text
解释：
- 仿射空间：Spec k[x₁, ..., xₙ]
- 射影空间：Proj k[x₀, ..., xₙ]
- Proj 构造「自动」处理了齐次性和「在无穷远处」的行为
```

### 13.2.3 仿射坐标卡

> **定义 13.2.3（仿射坐标卡）**
> 
> 射影空间 Pⁿ 可以被 n+1 个仿射空间覆盖：
> 
> ```
> U_i = { [x₀ : ... : xₙ] | x_i ≠ 0 } ≅ Aⁿ
> ```
> 
> 在 U_i 上，我们有仿射坐标：
> 
> ```
> (x₀/x_i, x₁/x_i, ..., x̂{x_i}/x_i, ..., xₙ/x_i)
> ```
> （其中 x̂{x_i} 表示去掉该分量）

```text
例子：在 P² 中
- U₀ = {[1 : y : z]} ≅ A²，坐标 (y/z, 1/z) → (y, z)
- U₁ = {[x : 1 : z]} ≅ A²，坐标 (x/z, z/z) → (x, z)
- U₂ = {[x : y : 1]} ≅ A²，坐标 (x/1, y/1) → (x, y)

每张卡片是一个仿射平面！
重叠部分用坐标变换粘起来！
```

## 13.3 射影簇

### 13.3.1 齐次多项式

> **定义 13.3.1（齐次多项式）**
> 
> k[x₀, ..., xₙ] 中的**齐次多项式**是所有项次数相同的多项式：
> 
> ```
> f(x₀, ..., xₙ) = Σ a_{i₀...iₙ} x₀^{i₀}...xₙ^{iₙ}
> ```
> 
> 其中 i₀ + ... + iₙ = d（固定次数 d）。

```text
例子：
- x² + y² + z²（2次齐次）
- x³ + 2xyz + y³（3次齐次）
- x + 1（非齐次，不能直接用于射影！）
```

### 13.3.2 射影簇的定义

> **定义 13.3.2（射影簇）**
> 
> **射影簇（Projective Variety）** 是射影空间 Pⁿ 中的闭子集，定义为某个齐次理想 I 的零点集：
> 
> ```
> V_+(I) = { [x₀ : ... : xₙ] ∈ Pⁿ | f(x₀, ..., xₙ) = 0, ∀f ∈ I }
> ```
> 
> 其中 I 是齐次理想。

```text
例子：
- 射影直线 P¹ = V_+(0)（整个空间）
- 射影平面曲线 V_+(f)，f 是 2 次或更高次齐次多项式
- 椭圆曲线：V_+(y²z - x³ - axz² - bz³)（3次平面曲线）
```

### 13.3.3 射影簇的例子

```text
例子1：射影直线
- P¹ = V_+(0) ⊂ P¹（整个空间）

例子2：二次曲线（圆锥曲线）
- V_+(x² + y² - z²) ⊂ P²
- 这是一个椭圆曲线（当非奇异时）！

例子3：扭结三次曲线
- V_+(y²z - x³ - axz² - bz³) ⊂ P²
- 这就是「椭圆曲线」！

例子4：费马曲线
- V_+(xⁿ + yⁿ + zⁿ) ⊂ P²
- 当 n ≥ 4 时，这是光滑曲线！
```

## 13.4 射影空间的闭集

### 13.4.1 Zariski 拓扑

> **定义 13.4.1（射影空间的 Zariski 拓扑）**
> 
> 在射影空间 Pⁿ 中，**闭集**定义为：
> 
> ```
> V_+(I)，其中 I 是齐次理想
> ```
> 
> 或者等价为：闭集的补是主开集 D_+(f)。

```text
例子：
- V_+(0) = Pⁿ（全集）
- V_+(1) = ∅（空集）
- V_+(f) = 齐次多项式 f 的零点集
```

### 13.4.2 射影空间的不可约性

> **命题 13.4.2（射影空间的不可约性）**
> 
> 射影空间 Pⁿ 是不可约的（作为拓扑空间）。

```text
证明思路：
- Pⁿ 不能写成两个真闭子集的并
- 因为任何闭子集都包含在某个超平面中
- 但超平面不能覆盖整个空间！
```

## 13.5 射影空间上的层

### 13.5.1 射影空间的结构层

> **定义 13.5.1（射影空间的结构层）**
> 
> 在射影空间 Pⁿ = Proj k[x₀, ..., xₙ] 上，结构层 O_{Pⁿ} 定义为：
> 
> ```
> O_{Pⁿ}(D_+(f)) = k[x₀, ..., xₙ]_{(f)}
> ```
> 
> 其中 (f) 表示对分母中 f 的幂次不敏感的局部化。

```text
解释：
- D_+(f) = {点 | f(点) ≠ 0}
- 在这个开集上，函数「可以除以 f」
- 结构层给出了射影空间的代数结构！
```

### 13.5.2 可逆层与线丛

> **定义 13.5.2（可逆层/线丛）**
> 
> **可逆层（Invertible Sheaf）** 或 **线丛（Line Bundle）** 是局部同构于结构层的层。

```text
例子：丰富层 O(d)
- O(d) = 结构层的 d 次「扭」
- 在仿射卡 U_i 上：O(d)|_{U_i} ≅ O_{U_i}
- 截面：齐次多项式的「权重」
```

### 13.5.3 丰富层

> **定义 13.5.3（丰富层）**
> 
> 层 L 是**丰富的（Ample）**，如果存在正整数 m 使得 L^⊗m 是非常丰富层（给出闭浸入到射影空间）。

```text
几何直觉：
- 丰富层 = 「可以嵌入到射影空间」
- 正如向量丛的「截面」可以给出嵌入！
```

## 13.6 射影态射

### 13.6.1 Proj 构造

> **定义 13.6.1（Proj 构造）**
> 
> 给定分次环 S = ⊕_{d≥0} S_d，其中 S_0 是域 k，S 是有限生成 k-代数，
> 
> **Proj(S)** 是一个概形，局部同构于：
> 
> ```
> Spec S_{(f)}，对每个齐次生成元 f ∈ S_d
> ```

### 13.6.2 射影态射

> **定义 13.6.2（射影态射）**
> 
> 态射 f: X → Y 是**射影的（Projective）**，如果局部地 X ≅ Proj(⊕_{d≥0} I^d/I^{d+1})，其中 I 是某个理想层。

```text
几何直觉：
- 射影态射 = 「在射影空间中给出」
- proper（紧致）+ finite type = 射影态射
- 这确保了「好的」行为！
```

## 13.7 维数与次数

### 13.7.1 射影簇的维数

> **定义 13.7.1（射影簇的维数）**
> 
> 射影簇 V ⊂ Pⁿ 的**维数** dim V 是：
> - V 作为拓扑空间的 Krull 维数
> - 或者：函数域 k(V) 的超越次数

```text
例子：
- dim Pⁿ = n
- dim 曲线 = 1
- dim 曲面 = 2
- dim 点 = 0
```

### 13.7.2 次数

> **定义 13.7.2（射影簇的次数）**
> 
> 设 V ⊂ Pⁿ 是 d 维射影簇。V 的**次数** deg(V) 是：
> 
> ```
> deg(V) = H^{n-d} · V（超平面截面数）
> ```
> 
> 即 V 与 n-d 个一般超平面的交点个数。

```text
例子：
- 直线（1维）在 P² 中的次数 = 1
- 圆锥曲线（1维）在 P² 中的次数 = 2
- 椭圆曲线（1维）在 P² 中的次数 = 3

几何直觉：
- 次数 = 「degree」
- 次数 = 曲线与直线相交的点数！
```

## 13.8 射影簇的例子：平面曲线

### 13.8.1 平面曲线的定义

> **定义 13.8.1（平面曲线）**
> 
> **平面曲线（Plane Curve）** 是射影平面 P² 中的 1 维射影簇：
> 
> ```
> C = V_+(f) ⊂ P²
> ```
> 
> 其中 f 是齐次多项式，次数为 d。

### 13.8.2 平面曲线的分类

```text
按次数 d 分类：

d = 1：直线
- 最简单的曲线
- 次数 = 1

d = 2：圆锥曲线
- 椭圆、抛物线、双曲线
- 次数 = 2
- 典型方程：ax² + bxy + cy² + dxz + eyz + fz² = 0

d = 3：三次曲线
- 椭圆曲线
- 典型方程：y²z = x³ + axz² + bz³

d ≥ 4：一般平面曲线
- 亏格公式：g = (d-1)(d-2)/2
```

---

## 13.9 曲线论初步

### 13.9.1 亏格

> **定义 13.9.1（平面曲线的亏格）**
> 
> 平面代数曲线的**亏格（Genus）** g 是：
> 
> ```
> g = (d-1)(d-2)/2
> ```
> 
> 其中 d 是曲线的次数。

```text
例子：
- d = 1（直线）：g = 0（有理曲线）
- d = 2（圆锥曲线）：g = 0（有理曲线）
- d = 3（椭圆曲线）：g = 1（椭圆曲线！）
- d = 4：g = 3
- d = 5：g = 6
```

### 13.9.2 有理曲线 vs 椭圆曲线

```text
- 有理曲线（g = 0）：同构于 P¹
- 椭圆曲线（g = 1）：一种特殊的 1 维阿贝尔簇
- 高亏格曲线（g ≥ 2）：复杂的有理曲线模空间
```
