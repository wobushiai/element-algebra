# 第七章：域扩张 —— 数的延伸

> **本章导读**：从有理数 ℚ 出发，我们可以添加根号、π、e 等得到更大的域。域扩张是研究「哪些数可以多项式表示」的工具，也是理解代数曲线的分类的基础。

---

## 7.1 域：能做完所有四则运算

### 7.1.1 域的定义

> **域（Field）** 是满足以下条件的环：
> - 交换环
> - 非零元都可逆（即 (F\{0}, ×) 是阿贝尔群）

```text
简单说：域就是「可以加减乘除」的集合！

例子：
- ℚ（有理数）
- ℝ（实数）
- ℂ（复数）
- ℤₚ（素数阶有限域）
```

### 7.1.2 域的例子

```text
不是域的例子：
- ℤ（整数）—— 2 的乘法逆元不是整数
- ℤ₆（模6）—— 2×3=0，所以 2 没有逆元

是域的例子：
- ℚ —— 所有非零有理数都有倒数
- ℝ —— 所有非零实数都有倒数
- ℂ —— 所有非零复数都有倒数
- ℤ₂ = {0, 1}，1+1=0，乘法：1×1=1
- ℤ₃ = {0,1,2}，3是素数，所以是域
- ℤₚ（p 为素数）—— 有限域
```

---

## 7.2 域扩张：从小域到大域

### 7.2.1 定义

> 若域 K 包含子域 F，则称 **K 是 F 的域扩张**，记作 K/F 或 K ⊇ F。

```text
例子：
- ℝ ⊇ ℚ（实数是有理数的扩张）
- ℂ ⊇ ℝ（复数是实数的扩张）
- ℚ(√2) ⊇ ℚ（添加 √2 得到的域）
```

### 7.2.2 扩张的次数

> **[K:F]** = K 作为 F 上的向量空间的维数

```text
例子：
- [ℝ:ℚ] = ∞（实数作为有理数上的向量空间是无限维的）
- [ℚ(√2):ℚ] = 2（基：1, √2）
- [ℂ:ℝ] = 2（基：1, i）
```

---

## 7.3 代数元与超越元

### 7.3.1 代数元

> 若 α 满足某个非零多项式 f(x) ∈ F[x]，则称 α 是 **F 上的代数元**。

```text
例子：
- √2 在 ℚ 上是代数的（满足 x² - 2 = 0）
- i 在 ℚ 上是代数的（满足 x² + 1 = 0）
- √[3]{2} 在 ℚ 上是代数的（满足 x³ - 2 = 0）
```

### 7.3.2 超越元

> 若 α 不满足任何非零多项式，则称 α 是 **F 上的超越元**。

```text
例子：
- π 在 ℚ 上是超越的（Lindemann-Weierstrass 定理）
- e 在 ℚ 上是超越的
- √2 + π 在 ℚ 上是超越的吗？（未知！）
```

### 7.3.3 代数扩张

> 若 K/F 中每个元素都是 F 上的代数元，则称 K/F 是 **代数扩张**。

```text
- ℚ(√2)/ℚ 是代数扩张（每个元素都可以写成 a + b√2）
- ℝ/ℚ 不是代数扩张（π 是超越的）
```

---

## 7.4 单扩张：添加一个元素

### 7.4.1 生成域

> 从域 F 添加一个元素 α 得到的最小域，记作 F(α)。

```text
ℚ(√2) = {a + b√2 | a, b ∈ ℚ}
ℚ(i) = {a + bi | a, b ∈ ℚ}
ℚ(√2, √3) = {a + b√2 + c√3 + d√6 | a,b,c,d ∈ ℚ}
```

### 7.4.2 极小多项式

> α 在 F 上的 **极小多项式** 是满足 f(α) = 0 的首一不可约多项式。

```text
例子：
- √2 在 ℚ 上的极小多项式：x² - 2
- i 在 ℚ 上的极小多项式：x² + 1
- √[3]{2} 在 ℚ 上的极小多项式：x³ - 2
```

### 7.4.3 次数公式

> 若 α 的极小多项式次数为 n，则 [F(α):F] = n。

```text
例子：
- √2 的极小多项式 x²-2 次数=2 → [ℚ(√2):ℚ] = 2
- i 的极小多项式 x²+1 次数=2 → [ℚ(i):ℚ] = 2
- √[3]{2} 的极小多项式 x³-2 次数=3 → [ℚ(√[3]{2}):ℚ] = 3
```

---

## 7.5 有限生成扩张

### 7.5.1 有限生成扩张

> 若 K = F(α₁, α₂, ..., αₙ)，则 K/F 是 **有限生成的**。

```text
例子：ℚ(√2, √3) = ℚ(√2)(√3)
- [ℚ(√2):ℚ] = 2
- [ℚ(√2, √3):ℚ(√2)] = 2（√3 在 ℚ(√2) 上满足 x²-3）
- [ℚ(√2, √3):ℚ] = 2×2 = 4
```

### 7.5.2 塔公式

> **塔公式**：若 F ⊂ K ⊂ L，则 [L:F] = [L:K] × [K:F]

```text
例子：
ℚ ⊂ ℚ(√2) ⊂ ℚ(√2, √3)
[ℚ(√2):ℚ] = 2
[ℚ(√2, √3):ℚ(√2)] = 2
[ℚ(√2, √3):ℚ] = 2×2 = 4
```

---

## 7.6 代数闭域

### 7.6.1 代数闭域的定义

> 若域 F 满足「每个非常数多项式都在 F 中有根」，则 F 是 **代数闭域**。

```text
等价说法：
- 每个非常数多项式都可以在 F 中完全分解为一次因式
- F 的代数扩张只有 F 本身

例子：
- ℂ 是代数闭域（代数基本定理）
- ℚ 不是代数闭域（x² + 1 = 0 无解）
- ℝ 不是代数闭域（x² + 1 = 0 无解）
```

### 7.6.2 代数闭包

> F 的 **代数闭包** F̄ 是包含 F 的最小代数闭域。

```text
例子：
- ℂ 是 ℝ 的代数闭包
- ℚ 的代数闭包是 ℚ̄（代数数域）
```

---

## 7.7 有限域

### 7.7.1 有限域的存在性

> **定理**：存在 pⁿ 阶的有限域当且仅当 p 是素数，n 是正整数。

### 7.7.2 有限域的结构

> 任何 pⁿ 阶有限域都同构于 GF(pⁿ) = ℤₚ[x]/(f(x))，其中 f 是 n 次不可约多项式。

```text
例子：
- GF(2) = ℤ₂
- GF(4) = ℤ₂[x]/(x²+x+1)（x²+x+1 在 ℤ₂ 上不可约）
- GF(8) = ℤ₂[x]/(x³+x+1)
```

---

## 7.8 伽罗瓦理论初步

### 7.8.1 伽罗瓦群

> 若 K/F 是有限可分扩张，则 **伽罗瓦群** Gal(K/F) 是 K 的 F-自同构群。

```text
例子：ℚ(i)/ℚ
Gal(ℚ(i)/ℚ) = {恒等, 共轭}
- 恒等：a + bi → a + bi
- 共轭：a + bi → a - bi
```

### 7.8.2 伽罗瓦对应

> **伽罗瓦对应**：K/F 的中间域与 Gal(K/F) 的子群之间存在反序双射。

> 这是**伽罗瓦理论**的核心！它把域扩张的问题转化为群论问题。

---

## 7.9 域扩张在代数几何中的应用

> 💡 **伏笔**：域扩张在代数几何中非常重要！
> 
> - **函数域**：代数曲线的函数域 K(C) 是 ℚ 或 k 的有限生成扩张
> - **可分扩张**：研究曲线的奇点
> - **代数闭域**：我们通常在代数闭域（如 ℂ, k̄）上研究代数几何
> - **剩余域**：研究点在特殊化后的行为

> 在后续章节中，我们将看到：
> - 射影直线 P¹ 的函数域 ≅ ℂ(t)
> - 椭圆曲线的函数域是二次扩张
