# 附录 C：常用符号与记号

> 本附录汇总了本书使用的常用数学符号，便于快速查阅。

---

## 基础集合符号

| 符号 | 含义 | 示例 |
|------|------|------|
| ℕ | 自然数 | ℕ = {1,2,3,...} 或 {0,1,2,...} |
| ℤ | 整数 | ℤ = {..., -2, -1, 0, 1, 2, ...} |
| ℚ | 有理数 | ℚ = {a/b : a,b ∈ ℤ, b ≠ 0} |
| ℝ | 实数 | 实数轴上的所有点 |
| ℂ | 复数 | ℂ = {a + bi : a,b ∈ ℝ, i² = -1} |
| 𝔽_q | 有限域 | q 个元素的有限域 |
| k | 代数闭域 | 通常指 ℂ 或 𝔽̄_q |

## 环论符号

| 符号 | 含义 | 示例 |
|------|------|------|
| R | 环 | ℤ, k[x], k[[x]] |
| R[x] | 多项式环 | ℤ[x], ℂ[x,y] |
| R_f | 局部化 | ℤ_2 = {a/2^n} |
| R_m | 在极大理想m处的局部化 | (k[x])_(x) |
| Spec(R) | 环R的谱 | Spec ℤ = {素理想} |
| Proj(R) | 分次环的射影谱 | Proj k[x₀,...,x_n] = Pⁿ |
| Hom_R(M,N) | R-模同态 | Hom_ℤ(ℤ, ℤ/n) = ℤ/n |
| M ⊗_R N | 张量积 | ℤ/n ⊗_ℤ ℤ/m = ℤ/gcd(n,m) |
| Tor_i^R(M,N) | Tor函子 | 测量张量积的偏差 |

## 理想与商环

| 符号 | 含义 | 示例 |
|------|------|------|
| ⟨a₁,...,aₙ⟩ | 生成的理想 | ⟨2,x⟩ ⊂ ℤ[x] |
| (a) | 主理想 | (2) = 2ℤ |
| I ∩ J | 理想的交 | ⟨x⟩ ∩ ⟨y⟩ = ⟨xy⟩ |
| I + J | 理想的和 | ⟨x⟩ + ⟨y⟩ = ⟨x,y⟩ |
| I·J | 理想的乘积 | ⟨2⟩·⟨3⟩ = ⟨6⟩ |
| √I | 根理想 | √⟨x²⟩ = ⟨x⟩ |
| R/I | 商环 | ℤ/6ℤ ≅ ℤ_6 |
