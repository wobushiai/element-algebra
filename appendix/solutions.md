## 第一章：集合与逻辑

### 基础题答案

**1. 用列举法表示：**

- {x | x是正方形且边长小于5}
  = {边长为1,2,3,4的正方形}

- {x ∈ ℕ | x是偶数且 x < 10}
  = {2, 4, 6, 8}

**2. 已知 A = {1, 2, 3}，B = {3, 4, 5}：**

- A ∪ B = {1, 2, 3, 4, 5}
- A ∩ B = {3}
- A \ B = {1, 2}

**3. 判断真假：**

- ∅ ⊆ {1, 2} = 真（空集是任何集合的子集）
- {1} ∈ {{1}, {2}} = 真（{1}是集合的元素）

### 进阶题答案

**4. 证明：A ∩ ∅ = ∅**

证明：
- 任取 x ∈ A ∩ ∅
- 则 x ∈ A 且 x ∈ ∅
- 但 x ∈ ∅ 不可能
- 所以 A ∩ ∅ = ∅

**5. 化简：(A ∩ B) ∪ (A ∩ Bᶜ)**

解：
- = A ∩ (B ∪ Bᶜ)
- = A ∩ U
- = A

## 第二章：数的世界

### 基础题答案

**1. 判断所属集合：**

- -5 ∈ ℤ, ℚ, ℝ, ℂ
- 3.14 ∈ ℚ, ℝ, ℂ
- √(-4) = 2i ∈ ℂ
- 0 ∈ ℕ?, ℤ, ℚ, ℝ, ℂ（取决于ℕ的定义）
- √2 ∈ ℝ, ℂ（无理数）

**2. 计算：**

- (2 + 3i) + (1 - i) = 3 + 2i
- (1 + i)(1 - i) = 1 - i² = 2
- i⁴ = (i²)² = (-1)² = 1

**3. 证明：任意两个有理数的和还是有理数**

证明：
- 设 a = p₁/q₁, b = p₂/q₂，其中 p₁,p₂ ∈ ℤ, q₁,q₂ ∈ ℤ\{0}
- a + b = (p₁q₂ + p₂q₁)/(q₁q₂)
- 分子分母都是整数，且分母 ≠ 0
- 所以 a + b ∈ ℚ

## 第三章：多项式

### 基础题答案

**1. f(x) = 3x⁴ - 2x² + x - 1**

- 次数 = 4

计算：
- f(2) = 2³ - 8 = 8 - 8 = 0

## 第五章：群论

### 基础题答案

**1. 验证 {1, -1} 在乘法下成群**

验证：
- 封闭：1·1=1, 1·(-1)=-1, (-1)·(-1)=1
- 单位元：1
- 逆元：1⁻¹=1, (-1)⁻¹=-1
- 结合律：显然

所以 {1, -1} 在乘法下成群。

**2. S₃ 的所有元素**

S₃ = {e, (12), (13), (23), (123), (132)}

**3. ℤ₈ 中由 2 生成的循环子群**

ℤ₈ = {0,1,2,3,4,5,6,7}

2的幂：
- 2·1 = 2
- 2·2 = 4
- 2·3 = 6
- 2·4 = 0

所以生成的子群 = {0, 2, 4, 6}
